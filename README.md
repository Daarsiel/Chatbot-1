# Chatbot-1
My first AI chatbot inspired from the first 10 mins of Harvard's 2026 CS50x course.

It was a pretty basic demo of API calling, and they used OpenAI. However, apparently OpenAI has paid API keys, so I tried
an alternate free option which is groq. But because I felt like I modified my code so much already, why not make my own chatbot instead
by implementing the model? 

So I did dynamic calling, checked streamlit(it was a pain and I had to keep on asking claude how to implement it correctly, I do 
wanna experiment more with that library). 

It ended up being the most basic chatbot. But I am proud of it because this is my first time making something like this, and also
uploading it as a repo. I think my journey just began.

I do plan on adding a "answer in one word" and a "cat personality" mode just like it was done as an example in the CS50x course.
